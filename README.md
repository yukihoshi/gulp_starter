# my first gulp task

##Installation
```bash
git clone https://github.com/yukihoshi/sp_site.git MyApp
cd MyApp
npm install
npm run gulp
```

##Features
This allows you to use...
- Gulp
- Less
- Sass (optional)
- Autoprefixr
- BrowserSync
- Imagemin
- Bootstrap
- jQuery


##Usage
edit and add files in the src directory, the final production goes to the dist directory.
